# Sistem Monitörü (Dashboard)

Yerel sistem kaynaklarını izlemek için basit bir dashboard/script.

Dosya
- `system_monitor.py` — sistem verilerini toplayan ve gösteren script.

Gereksinimler
- Python 3.x
- `psutil` gibi sistem bilgisi kütüphaneleri olabilir; script içindeki importlara bakın.

Kullanım
- `python system_monitor.py`

Not
- Script'i bir uzaktan izleme amacıyla kullanacaksanız güvenli erişim ve kimlik doğrulamayı uygulayın.
