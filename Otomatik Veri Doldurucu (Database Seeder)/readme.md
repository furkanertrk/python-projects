# Otomatik Veri Doldurucu (Database Seeder)

Veritabanı için sahte/örnek veri oluşturmaya yarayan araç.

Yapı
- `src/database_seeder.py` — ana script
- `data/` — örnek/veri dosyaları

Gereksinimler
- Python 3.x
- `requirements.txt` içeriği varsa yükleyin: `pip install -r requirements.txt`

Kullanım
- Proje kökünden çalıştırın: `python src/database_seeder.py`
- Script içindeki parametreler ve bağlantı ayarlarını kontrol edin.

Not
- Gerçek veritabanlarında çalıştırmadan önce yedek alın.
- Üretilen sahte verilerin hassas verilerle karışmadığından emin olun.
