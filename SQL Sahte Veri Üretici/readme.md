# SQL Sahte Veri Üretici

Veritabanları için örnek/sahte veri üretmeye yarayan script.

Dosya
- `sql_generator.py`

Gereksinimler
- Python 3.x
- Gerekli kütüphaneler script başındaki importlara bağlıdır.

Kullanım
- `python sql_generator.py`
- Script içindeki parametreler (kayıt sayısı, tablo yapısı vb.) için dosyayı inceleyin.

Not
- Gerçek veritabanlarında çalıştırmadan önce mutlaka yedek alınız.
- Üretilen verilerin gizlilik gereksinimleri ve GDPR/yerel düzenlemelerle uyumlu olduğundan emin olun.
