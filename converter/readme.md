# Converter

Dosya dönüşümü yapan basit bir script içeren klasör.

Dosya
- `converter.py` — ana dönüşüm scripti.

Gereksinimler
- Python 3.x
- Gerekli kütüphaneler script içindeki importlara göre değişir.

Kullanım
- `python converter.py`
- Script içindeki yorumları ve kullanım bilgisini kontrol edin; CLI argümanları olabilir.

Not
- Hangi formatlar desteklendiğini öğrenmek için script'i açıp okumanızı öneririm.
