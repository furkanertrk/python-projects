# Local Network File Transfer

Yerel ağ üzerinde dosya aktarımı sağlayan basit bir araç.

Dosya
- `pydrop.py` — dosya transferi scripti.

Gereksinimler
- Python 3.x
- Gerekli kütüphaneler script başındaki importlara bağlıdır.

Kullanım
- `python pydrop.py`
- Klasör içindeki script açıklamalarını okuyun; CLI argümanları veya sunucu/istemci modları olabilir.

Güvenlik
- Bu aracı yalnızca güvenli ve izinli ağlarda kullanın.
- Açık portlar ve yetkisiz erişim güvenlik riskleri barındırır.
