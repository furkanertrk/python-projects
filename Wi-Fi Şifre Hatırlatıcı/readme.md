# Wi-Fi Şifre Hatırlatıcı

Kaydedilmiş Wi-Fi parolalarını hatırlamaya yardımcı olabilecek bir script.

Dosya
- `wifi_sifre_hatirlatici.py`

Gereksinimler
- Python 3.x
- Platforma özel erişim gereksinimleri olabilir (Windows: yönetici hakları vb.).

Uyarı
- Parolaları görüntüleme veya çıkartma işlemi izinsiz yapıldığında gizlilik ihlali ve yasal sorunlara yol açabilir.
- Bu aracı yalnızca kendi cihazlarınız ve izinli ortamlar için kullanın.
