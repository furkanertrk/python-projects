# Ağdaki Cihazları Listeleme (Network Scanner)

Basit bir ağ tarayıcı/cihaz listeleme scripti.

Özet
- Dosya: `network-scanner.py`
- Amaç: Aynı yerel ağdaki cihazları keşfetmek ve temel bilgiler almak.

Gereksinimler
- Python 3.x
- Gerekirse: `scapy` veya benzeri kütüphaneler (script içindeki importları kontrol edin).

Kullanım
- Temel: `python network-scanner.py`
- Script içindeki yorumlar ve komut satırı argümanlarını kontrol edin; parametreler proje tarafından sağlanır.

Notlar
- Bu araç yalnızca kendi ağınız veya izinli olduğunuz ağlarda kullanılmalıdır.
- İzinsiz ağ taraması yasal sorunlara yol açabilir. Lütfen yerel yasaları ve ağ sahibinin izinlerini kontrol edin.
