# Port Tarayıcı

Ağdaki bir hedefin açık portlarını kontrol etmeye yarayan script.

Dosya
- `port-tarama.py`

Gereksinimler
- Python 3.x

Kullanım
- `python port-tarama.py <hedef_ip>`
- Script içinde yer alan ek argümanları kontrol edin.

Uyarı
- Port taraması etik ve yasal kurallara tabidir. Yalnızca sahibi olduğunuz veya izin aldığınız sistemlerde tarama yapın.
