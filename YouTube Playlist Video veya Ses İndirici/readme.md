# YouTube Playlist Video veya Ses İndirici

YouTube çalma listelerinden video veya ses indirmeye yönelik bir yardımcı script.

Dosya
- `yt playlist indirici.py`

Gereksinimler
- Python 3.x
- Muhtemelen `pytube`, `yt-dlp` veya benzeri kütüphaneler gereklidir; script içindeki importlara bakın.

Uyarı ve Telif Hakları
- Bu aracı yalnızca telif hakkı sahibinin izni olduğu içerikler için veya yasal olarak izin verilen kullanım durumlarında kullanın.
- YouTube hizmet şartlarına ve telif hakkı kanunlarına uyun.

Kullanım
- `python "yt playlist indirici.py" <playlist_url>` gibi genel bir komut beklenebilir; kesin argümanlar için script içeriğini kontrol edin.

Not
- Sorularınız varsa script dosyasını açıp mevcut parametreleri inceleyin.
